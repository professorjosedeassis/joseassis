Linux Database Server

Este tutorial ensina a instalar, configurar e gerenciar o banco de dados MySQL no Linux. O principal objetivo é implementar de forma prática um servidor de banco de dados para uso na rede local em sistemas comercias de lojas, supermercados, escritórios, oficinas, etc. Isto pode trazer uma oportunidade de negócio, pois além de gratuito o servidor de banco de dados baseado no Linux não requer um hardware potente, atendendo clientes comerciais de pequeno ou médio porte. Esta playlist também ensina a trabalhar com um banco de dados relacional e com os fundamentos da linguagem SQL. Estas competências são muito importantes para estudantes e profissionais das áreas de infraestrutura e também das áreas de desenvolvimento de sistemas.
